# 🌳 AI Based Park Surveillance System

### Infosys Springboard Internship Project

This project is an **AI-Based Intelligent Video Surveillance System** developed for **Activity Recognition and Security Management in Public Parks**.
The system uses **Computer Vision and Deep Learning techniques** to detect human activities and monitor suspicious behaviour in park environments.

---

## 🚀 Project Overview

Public parks require constant monitoring to ensure safety and prevent illegal or suspicious activities.
This project provides an **AI-powered surveillance solution** that automatically analyzes video feeds and detects activities.

The system helps security teams monitor parks efficiently using **AI-based video analysis**.

---

* Python

* Computer Vision

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Madhu9302/Infosys-Springboard-AI-park-surveillance.git
```

Go to the project folder:

```
cd Infosys-Springboard-AI-park-surveillance
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Run the main application:

```
python app.py
```

For Streamlit interface:

```
streamlit run streamlit_app/app.py
```
